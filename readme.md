# Praktikum-I
Projekt pri predmetu Praktikum I

# Information

Projekt Praktikum-1 je spletna aplikacija, kjer so uporabljene tehnologije:

  1. PHP 7.2.10
  2. HTML, CSS
  3. SQL (phpmyadmin)

Uporabnik lahko:

  1. Se registrira
  2. Se prijavi
  3. Naročilo preveri in potrdi

Admin lahko:

  1. Pregleda naročila
  2. Izbriše naročilo
